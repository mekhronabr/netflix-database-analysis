# Анализ датасета Netflix

Программное обеспечение для анализа каталога Netflix.  
Курсовой проект HSE, «Python в науке о данных».

**Автор:** Бободжанова Мехрона Рахимовна

## Возможности

- просмотр и фильтрация контента;
- управление справочниками;
- текстовые и графические отчёты;
- аналитика по странам и динамике выпуска;
- поиск похожего контента (TF-IDF);
- сетевой анализ актёров и режиссёров.

## Запуск

```bash
cd Work/Scripts
python main.py
```

При первом запуске CSV из `Work/Data/` автоматически конвертируются в `.pkl`.

## Установка зависимостей

```bash
pip install -r requirements.txt
```

Рекомендуется виртуальная среда conda (Python 3.10+).

## Структура проекта

```
Work/
  Data/          — исходные CSV и бинарные .pkl
  Library/       — бизнес-логика и отчёты
  Notes/         — руководства пользователя и разработчика
  Scripts/       — GUI и точка входа main.py
  Output/        — экспорт таблиц
  Graphics/      — экспорт графиков
```

## Документация

Документация размещена в `Work/Notes/` (каталог **Notes** по ТЗ)
